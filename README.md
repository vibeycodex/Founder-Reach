# FounderReach

Autonomous VC meeting agent built on the TinyFish Web Agent API.

## What it does
Founders waste hours cold emailing investors who never respond. FounderReach sends a TinyFish web agent to navigate investor profiles across personal websites, firm pages, and Twitter, find public booking links, verify live slot availability, and surface verified investors ready to book — in seconds.

## Live Demo
https://founderreach.netlify.app/

## How it works
1. Founder describes their startup and selects investment stage and number of VCs
2. TinyFish agent navigates investor profiles, firm sites, and Twitter
3. Agent locates public booking links and confirms live availability
4. Verified results stream back to the dashboard via SSE
5. Founder views investor profile, focus, next available slot, and opens booking page directly
6. Meetings are plotted on a built-in calendar view for tracking

## Built with
- TinyFish Web Agent API (SSE endpoint)
- Vanilla HTML / CSS / JavaScript
- Tailwind CSS
